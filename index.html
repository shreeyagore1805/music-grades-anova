Hypotheses

H₀ — Null hypothesis
Mean grades are equal across all music groups. Music type has no significant effect on grades.
H₁ — Alternate hypothesis
At least one music group has a significantly different mean grade. Music type does affect grades.

How the study works

1
Collect data via Google Form
Students fill in study habits — hours studied, music type, last exam grade, and course stream.
2
Data loads live into Python
Responses auto-export to Google Sheets. Python pulls a live CSV from the sheet at runtime.
3
Run One-Way ANOVA
Test whether mean grade differs significantly across groups: Classical, Lo-fi, Pop, and No music.
4
Post-hoc analysis with Tukey HSD
If ANOVA is significant (p < 0.05), Tukey's test identifies which specific pairs of groups differ.

Decision rule

If p-value < 0.05
Reject H₀ — music type has a statistically significant effect on grades. Run Tukey HSD to find which groups differ.
If p-value ≥ 0.05
Fail to reject H₀ — no significant difference found in this sample.
